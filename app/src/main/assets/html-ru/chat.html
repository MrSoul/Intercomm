Чат

Ник нужно указать в Настройках.
Смайлы можно выбирать нажав соответвующую кнопку,при нажати на изображение будет вставлент текстовый код смайла.
Можно вставить ссылку с текущими координатами - для этого должен быть включен GPS. Втавка производится выбором соответвующего пункта в верхнем меню.
История чата сохраняется при закрытии программы и может быть очищенна соответвующим пунктом в верхнем меню.
Максимальная длина сообщения 70 символов(нужно проверять).
Проверка наличия сообщений производится раз в секунду.
