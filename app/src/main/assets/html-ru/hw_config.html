Аппаратная конфигурация


Эта секция настроек для работы напрямую с оборудыванием для работы программы на неподдерживаемых моделях телефонов без пересборки приложения(номера строк и описание их значений):
  1. Имя устройства - показывается во время включения(текст)
  2. порт на котором модуль например /dev/ttyMT1
  3. устройство для ioctl(зарезервировано и не используется еще) может быть /dev/null
  4. Команда ВКЛючения питания(может быть echo или ioctl) например echo "-w=75: 0 0 1 1 0 1 0" > /sys/class/misc/mtgpio/pin
  5. Команда ВЫКЛючения питания(может быть echo или ioctl) например echo "-w=75: 0 0 0 0 0 1 0" > /sys/class/misc/mtgpio/pin
  6. Команда установки режима спикера
  7. Команда установки режима наушников
  8. Команда нажатия кнопки(Включения передачи)
  9. Команда отпускания кнопки(Выключения передачи)
  10. Версия модуля рации(текст)
  11. Минимальная частота(точка отделяет целую часть) например 400.0
  12. Максимальная частота(точка отделяет целую часть) например 480.0
  13. Максимальная громкость(целое) например 8
  14. Максимальное число тонов(целое) например 39
  15. Использовать передающие тоны (Булевое) "true" или "false" без кавычек
  16. Строка устанавливается в "(auto)" чтобы показать автоматическую конфигурацию и должна быть удалена для использования ручной
